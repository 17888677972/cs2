# 项目介绍
该项目旨在实现CS2的外部读写功能，提供多种辅助功能，包括位置报点、热能透视以及模拟鼠标操作自瞄功能。**请注意**，此版本仅供学习参考和功能研究使用。

## 功能列表
- **位置报点**：实时获取并报告目标位置
- **热能透视**：启用后可通过热成像效果显示目标
- **模拟自瞄**：通过模拟鼠标操作实现自动瞄准

# 使用说明

- **F1**：启用/关闭热能透视
- **F2**：启用/关闭自瞄功能
- **F3**：启用/关闭位置报点
